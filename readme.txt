== Description ==
A widget for reserving hotel rooms.

== Installation ==

    Download WP Hotel Reservation widget.
    Upload the \'wp-hotel-reservation-widget\' directory to your \'/wp-content/plugins/\' directory, using your favorite method (ftp, sftp, scp, etc...)
    Activate WP Hotel Reservation widget from your Plugins page.)
   You can see a new WP Hotel Reservation widget in Appearance->widgets, drag and drop to your sidebar
